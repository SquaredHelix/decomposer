# Decomposer
Decomposer is a small command-line tool designed to render files generated by Icon Composer into a flat png.

By default, Icon Composer only exports liquid glass versions of your icon. This is fine for liquid glass supported devices, but will look out of place for other software versions. 

With this tool you can render your design into a transparent canvas, or onto your own supplied 1024px background. This way it can be used to generate assets that work better for non liquid glass devices.

> [!WARNING]
> Advanced Icon Composer features like gradients and fills are not yet implemented.
>
> Ideally effects should be applied to the source files instead

```bash
Usage: decomposer.exe [-o <output>] [-b <background>] [-p] [--] <input>

Renders Icon Composer files

Positional Arguments:
  input             input path

Options:
  -o, --output      output path
  -b, --background  background path
  -p, --print-tree  print parsed tree
  --help, help      display usage information
```

## Installation
The simplest way to install the tool is through cargo.
This should also handle adding to path.
```yml
cargo install --git https://github.com/SquaredHelix/decomposer
```
