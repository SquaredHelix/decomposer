# Decomposer
Decomposer is a small command-line tool designed to render files generated by Icon Composer into a flattened format.

By default, Icon Composer only supports exporting liquid glass versions of your assets. This looks good, but might not be ideal for backwards compatibility.

> [!WARNING]  
> Advanced Icon Composer features like gradients and fills are not yet implemented. 
> Ideally effects should be applied to the source files instead
